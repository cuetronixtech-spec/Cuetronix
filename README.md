# Cuetronix

Cuetronix is a multi-tenant club and venue management platform built with React, TypeScript, and Vite.

## Features

- Point of Sale (POS)
- Station & product management
- Customer portal & bookings
- Tournaments & scheduling
- Staff, expenses, and cash management
- Reports & analytics
- AI chat assistant
- Super admin dashboard

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
